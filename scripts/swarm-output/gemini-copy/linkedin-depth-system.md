# LinkedIn Post — Shibui Depth System

---

I built a system that rewrites my portfolio in real-time based on who's reading it.

Here's the problem: technical portfolios speak fluent engineer to fluent engineer. Recruiters land on "AI Orchestration Architect" and bounce in 7 seconds. They've got 100 other slots to fill.

So I built Shibui.

**What it does:**
- Scores every paragraph by complexity (readability + domain density + citation count)
- Generates simplified entry text using five rhetorical transformation functions
- Tags domain terms with inline definitions

**How it works:**
1. Homepage leads with plain language (Overview mode)
2. Click depth to see annotated text with term definitions (Standard)
3. Full academic mode for specialists who want the complete picture

**The technical flex:**
Five functions grounded in linguistic theory:
- Complexity Reduction (Chomsky's Transformational Grammar)
- Term Substitution (hypernym chains from vocabulary.json)
- Information Density Control (Shannon entropy scoring)
- Register Shift (Halliday's Systemic Functional Linguistics)
- Coherence Preservation (Rhetorical Structure Theory)
